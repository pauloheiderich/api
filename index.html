SOBRE

O projeto desenvolvido consiste em uma aplicação web

que permite consultar a Tabela FIPE, seu objetivo é permitir

aos usuários consultar valores de veículos

(carros, motos e caminhões) através da tabela FIPE oficial.

O tema foi escolhido pela utilidade prática da ferramenta

e pela disponibilidade de dados atualizados através de API pública.
